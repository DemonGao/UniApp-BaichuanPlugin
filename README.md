#阿里百川电商SDK插件

##1.使用注意
使用前请注意

* 本插件包提供阿里百川基础电商功能，包括：
  - 初始化
  - 登录授权（获得用户信息）
  - 退出登录
  - 跳转到淘宝店铺
  - 跳转到商品详情
  - 跳转到优惠券页面
  
* 支持在线云打包
* 使用时请替换阿里百川安全图片（V4版本，具体参加百川平台）。将安全图片放置在插件包下的ios根目录下。
* 由于HBuilderX对包大小的限制（40M），**插件仅支持Arm64架构**，即支持的最低iOS机型为iphone5s，5s一下机型不支持
